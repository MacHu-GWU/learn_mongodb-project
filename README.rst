Install
-------------------------------------------------------------------------------
First, Make sure you have `pip <https://pip.pypa.io/en/stable/>`_ installed. Then:

.. code-block:: console

	$ cd <path-to-learn_mongodb-project-directory>
	$ pip install . --upgrade
